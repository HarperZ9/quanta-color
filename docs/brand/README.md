# Build Color brand assets

- `build-color-mark.svg` — square logomark (gamut triangle + white point). Use for
  avatars, favicons, and small placements.
- `build-color-hero.svg` — wide banner used at the top of the project README.

Palette: the marks use the Project Telos indigo (`#4636e8`) with color-primary accents
(`#ff4d6d` / `#38d39f` / `#4d8bff`) on a near-black indigo ground (`#160c28`/`#0a0816`).
Assets are SVG so they scale cleanly; export to PNG only if a raster is required.
